# Jogo Plataforma 2D - Coleta de Diamantes

## Descrição

Este projeto consiste em um jogo de plataforma 2D desenvolvido na Unity. O jogador controla um personagem que deve explorar o cenário e coletar todos os diamantes espalhados pela fase.

O projeto foi desenvolvido com o objetivo de aplicar conceitos de desenvolvimento de jogos digitais, programação em C# e utilização da engine Unity.

## História

Um aventureiro parte em uma missão para recuperar diamantes mágicos perdidos pelo cenário. Para completar sua jornada, ele precisa explorar o ambiente, superar plataformas e coletar todos os diamantes encontrados pelo caminho.

## Mecânicas

* Movimentação para esquerda e direita
* Sistema de salto
* Coleta de diamantes
* Sistema de colisão
* Exploração de cenário em plataforma 2D

## Tecnologias Utilizadas

* Unity
* C#
* Assets disponibilizados pelo professor

## Objetivo

Coletar todos os diamantes presentes na fase para completar o desafio.

## Aprendizados

Durante o desenvolvimento deste projeto foram aplicados conceitos relacionados a:

* Jogos de plataforma 2D
* Programação orientada a objetos
* Scripts em C#
* Sistema de colisões
* Interatividade em jogos
* Utilização de sprites e assets na Unity
